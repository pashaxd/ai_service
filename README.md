# ai_service

Сервис для сравнения двух редакций `DOCX`/`PDF` и генерации JSON-отчёта по каждому пункту изменения.

## Без FastAPI: CLI

## Как запустить

1. Установить зависимости (например, через `pip`/`venv`).
2. (Опционально) Задать `POLZA_API_KEY` — если хочешь запускать LLM + web-search.

Можно запускать сравнение как утилиту без HTTP:

```bash
POLZA_API_KEY="..." compare-ai --file-a /path/old.pdf --file-b /path/new.pdf --run-agents
```

Если ключ не задавать, добавь `--run-agents` не будет выполняться LLM/веб-поиск (только diff+points).

## Ускорение (асинхронность)

В режиме `--run-agents` пункты (`points`) обрабатываются параллельно, а порядок в финальном JSON сохраняется.
Лимит параллельности задаётся переменной окружения `MAX_PARALLEL_POINTS` (по умолчанию `3`).
